SharedModel(
  (net_container): Sequential(
    (0): Linear(in_features=2, out_features=32)
    (1): ELU(alpha=1.0)
    (2): Linear(in_features=32, out_features=32)
    (3): ELU(alpha=1.0)
  )
  (policy_layer): Linear(in_features=32, out_features=1)
  (value_layer): Linear(in_features=32, out_features=1)
)
 ^ctNsx5yx

 net_container
是一个共享的前向神经网络部分，用于提取特征，结构如下：

输入维度：2（说明 observation 的维度是 2）

两个全连接层（Linear） + 激活函数（ELU）：

第一层：2 → 32

第二层：32 → 32

激活函数使用 ELU（Exponential Linear Unit），它在负区间不会完全为 0，常用于对抗梯度消失问题。 ^RX9A3kNz

policy_layer 和 value_layer
policy_layer：接收 net_container 的输出（32维），输出一个动作（1维），适用于连续动作空间的 actor。

value_layer：也是接收 net_container 的输出，输出一个值（1维），估计状态的 value。 ^f8iCnPoe
