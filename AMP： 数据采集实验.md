

## 1.实验参考
Experiment-free exoskeleton assistance via learning in simulation
![[Pasted image 20250731092837.png]]


研究人员开发了一个无需实验的基于仿真的学习框架，用于学习外骨骼控制策略，以协助多模态人类运动，包括步行、跑步和爬楼梯。他们利用了动力学感知的肌肉骨骼和外骨骼模型，以及数据驱动的强化学习，在仿真中学习了一种有效的控制策略，无需进行人体实验。

学习到的控制器被部署在一个定制的髋部外骨骼上，可以自动为不同的运动活动提供辅助。研究人员进行了实验，通过测量参与者在三种情况下（外骨骼辅助开启、外骨骼辅助关闭和无外骨骼）步行、跑步和爬楼梯时的代谢率，来评估外骨骼控制器的性能。

实验协议涉及收集8名参与者在三种运动活动中的代谢率和运动学数据。参与者信息总结在扩展数据表1中。

结果显示，与无外骨骼情况相比，外骨骼辅助可将代谢率降低最高达==**24.3%、13.1%和15.4%，分别用于步行、跑步和爬楼梯**== 。这些改善优于目前最先进的便携式外骨骼在同类活动中的性能。

### 🔄 数据转换说明

本项目通过 Python 脚本对原始 `.htr` 文件进行处理，提取以下内容：

- **躯干（Pelvis）**：三维平移 $[Tx, Ty, Tz]$（单位：米）与四元数旋转 $[Qx, Qy, Qz, Qw]$，并通过初始参考四元数 `q₀` 进行归一化与变换对齐。
    
- **其他关键关节**：旋转角度 $[Rx, Ry, Rz]$（单位：弧度），选定的关节包括左右大腿、左右小腿、双脚，部分角度加入了微调偏移以贴合实际骨骼结构。

## 🧠 AMP 模型训练所需的动作采集任务设计

为了训练**Adversarial Motion Priors（AMP）**模型以学习丰富的人体运动先验，特设计如下关键动作采集任务：

| 编号  | 任务名称          | 动作设计说明                                                                        |
| --- | ------------- | ----------------------------------------------------------------------------- |
| 1   | **匀速直线行走**    | 以稳定的速度沿直线路径自然步行，用于捕捉基础步态的稳定模式。                                                |
| 2   | **变速直线行走**    | 沿直线加速或减速行走，突出速度变化对身体动态的影响。                                                    |
| 3   | **S 型路径行走**   | 沿 S 形路径自由移动，包含转向与直线段，展示转向行为的自然过渡特征。                                           |
| 4   | **奔跑（含启动阶段）** | 从静止起步过渡至奔跑状态，涵盖启动、加速与稳定奔跑全过程。                                                 |
| 5   | **原地跳跃**      | 从静止状态起跳并自然落地，完整记录起跳准备、腾空与落地过程。                                                |
| 6   | **行走中跳跃**     | 在正常行走过程中突然起跳并继续行走，捕捉中断-恢复之间的运动协调性。                                            |
| 7   | **日常间歇性步态**   | **自然或特定情境下出现的、习惯性中断的步行行为**。它既不同于连续匀速的行走，也不完全属于异常步态，而是一种**带有明显节奏性或目的性停顿的运动模式** |

> ✅ **采集建议：**每种动作建议采集多个重复周期（如 3 次以上），以提升模型对时序变化与风格多样性的泛化能力。
