

在IsaacGymEnvs中，您可以通过多种方式查看训练过程中的数据。主要有以下几种方法：

## 使用TensorBoard查看训练数据

训练过程中的数据会自动记录到TensorBoard中，这是查看训练数据的主要方式：

1. 训练数据通过`RLGPUAlgoObserver`类记录 [1](#0-0) 

2. 数据会被记录到以下几个类别：
   - Episode/: 每个episode的统计数据
   - episode_cumulative/: 累积的episode数据
   - 其他直接从环境中记录的数据 [2](#0-1) 

3. 运行训练后，可以使用TensorBoard查看这些数据：
```bash
tensorboard --logdir runs
```

## 使用Weights & Biases (WandB)查看数据

您也可以使用WandB来可视化训练数据：

1. 确保已安装WandB：`pip install wandb`

2. 在训练命令中添加`wandb_activate=True`参数： [3](#0-2) 

3. 可以设置WandB的组、名称、实体和项目：
```bash
python train.py task=Ant wandb_activate=True wandb_entity=your_entity wandb_project=your_project
```

## 捕获训练视频

您可以在训练过程中自动捕获智能体的视频：

1. 添加`capture_video=True`参数： [4](#0-3) 

2. 可以上传视频到WandB： [5](#0-4) 

## 查看控制台输出

某些任务会在控制台直接打印训练统计信息，例如： [6](#0-5) 

## 自定义数据记录

如果需要记录特定数据，可以修改`post_physics_step`方法，该方法在每个物理模拟步骤后被调用： [7](#0-6) 

## 评估统计数据

对于某些任务，如AllegroKuka，还有专门的评估统计功能： [8](#0-7) 

Notes:
- 默认情况下，训练数据保存在`runs/EXPERIMENT_NAME`目录下，其中`EXPERIMENT_NAME`默认为任务名称
- 使用`v`键可以在训练过程中切换是否显示预览窗口，这可能会影响训练速度
- 对于PBT（Population Based Training）训练，有额外的数据记录在`pbt/`目录下
