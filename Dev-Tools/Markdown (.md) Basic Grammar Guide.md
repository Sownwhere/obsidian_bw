

## 1. Headings
Use `#` to `######` for headings.

```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

---

## 2. Paragraphs & Line Breaks
Paragraphs are plain text separated by a blank line.  
To force a **line break**, add **two spaces** at the end of a line.

```md
This is a paragraph.  
This line breaks after two spaces.
```

---

## 3. Text Styles

```md
*Italic* or _Italic_

**Bold** or __Bold__

***Bold & Italic***

~~Strikethrough~~
```

## 4. Lists

### Unordered List

```md
- Item 1
* Item 2
+ Item 3
```

### Ordered List

```md
1. First item
2. Second item
3. Third item
```

## 5. Links & Images

### Link:


### Image:

